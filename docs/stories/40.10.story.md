## Story

As a user navigating to stats,
I want the insights view to feel like entering a "trophy room" behind the three doors,
So that viewing my stats feels like a reward and integrates naturally with the door metaphor.

## Background

Currently, the stats view is a separate view with no thematic connection to doors. This story introduces the concept of the stats view as a space "behind" or "beyond" the doors — a personal trophy room. The transition from doors to stats should feel thematic, not just a screen swap.

## Acceptance Criteria

**Given** the user presses the insights key from the doors view
**When** the insights view opens
**Then** the transition includes a thematic header (e.g., "Your Trophy Room" or theme-appropriate title)
**And** the header style is consistent with the current door theme

**Given** the trophy room concept
**When** the insights dashboard renders
**Then** the overall layout evokes a "room" feel (bordered outer frame, interior panels)
**And** the visual treatment is distinct from the doors view but clearly part of the same application

**Given** each door theme (Classic, Modern, Sci-Fi, Shoji, Golden, Shadow)
**When** the trophy room renders
**Then** the room header and frame style match the active theme
**And** Sci-Fi gets a "Mission Briefing" feel, Golden gets "Treasury", Shoji gets "Tea Room" etc.

**Given** the user presses Esc from the trophy room
**When** returning to the doors view
**Then** the transition is instant (no exit animation required)

**Given** the existing insights view functionality
**When** the trophy room layout is applied
**Then** all existing stats panels, sparklines, bars, and fun facts remain functional
**And** the trophy room is a layout/theming enhancement, not a functional change

## Technical Notes

- This is primarily a View() rendering change — no new data or logic
- Theme-specific room titles stored in ThemeColors or a new ThemeStrings type
- Outer frame: full-width Lipgloss bordered panel with themed border style
- Consider subtle door-frame imagery in the outer border (theme dependent)
- Stretch goal: brief "doors sliding apart" transition animation (optional, time-permitting)

## Story

As a user viewing my insights,
I want to see a rotating "fun fact" line derived from my real session history,
So that the stats feel personal, surprising, and celebratory.

## Background

ThreeDoors captures extensive session data (doors opened, tasks completed, mood entries, door position selections, time to first door, etc.) but most of this data is never surfaced to the user. Fun facts mine this data for interesting personal observations.

Example fun facts:
- "You've opened 247 doors since you started!"
- "Your fastest session start: 0.3 seconds to pick a door"
- "You complete the most tasks on Wednesdays"
- "Focused mood = 4.2 tasks/session. That's your superpower!"
- "You tend to pick the left door 42% of the time"

## Acceptance Criteria

**Given** the user has session history data
**When** the insights view renders
**Then** a fun fact line appears at the bottom of the dashboard
**And** the fact is derived from real session data (not hardcoded)
**And** a different fact is shown each time the insights view is opened (random selection)

**Given** the user has minimal session history (< 3 sessions)
**When** the insights view renders
**Then** a generic welcome fact is shown (e.g., "Welcome! Keep going and we'll discover your patterns")
**And** facts requiring more data are excluded from the pool

**Given** the fun fact pool
**When** facts are generated
**Then** all facts use celebratory, observational language aligned with SOUL.md
**And** no fact uses judgmental, prescriptive, or guilt-inducing language
**And** no fact compares the user negatively to any standard or prior performance

**Given** the fun facts function
**When** tested
**Then** table-driven tests cover: sufficient data, minimal data, zero data, each fact type

## Technical Notes

- Create `GenerateFunFact()` function in `internal/core/` that takes PatternAnalyzer + SessionCounter data
- Each fact type has a minimum data threshold (e.g., "best day" needs >= 7 days of data)
- Use `math/rand` to select from available facts
- Display with a styled prefix (e.g., star icon or `*` bullet)
